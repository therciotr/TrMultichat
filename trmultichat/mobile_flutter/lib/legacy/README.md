# Legacy (código antigo / não usado pelo app atual)

Esta pasta contém arquivos **legados** que existiam antes da estrutura atual (Clean Architecture + MVVM + Riverpod).

- O app em uso hoje está em `lib/src/**` e o router ativo é `lib/src/router/app_router.dart`.
- Os arquivos em `lib/legacy/**` foram movidos **sem apagar** para reduzir duplicações e evitar conflitos.

> Observação: o conteúdo em `legacy` não é importado pelo router atual.
